### Installation
```
git clone https://github.com/vogelino/molecular-networks-p5-algorithm.git
cd molecular-networks-p5-algorithm
npm install
```

### Use
Run `gulp` and open `http://localhost:3000`

### Deploy to gh-pages
```
gulp deploy
```
